## 快捷键

C --> Ctrl

S --> Shift

M --> Alt

Cmd --> Command

### 常用操作

| 功能               | Windows  | Mac OS X |
|:-------------------|:---------|:---------|
| 当前页面下新建页面 | C-Return |          |
| 预览               | F5       |          |
| 生成 HTML 文件     | F8       |          |
| 置于顶层           | C-S-]    |          |
| 置于底层           | C-S-[    |          |
| 上移一层           | C-]      |          |
| 下移一层           | C-[      |          |
| 开关左侧功能栏     | C-M-[    |          |
| 开关右侧功能栏     | C-M-]    |          |

### 操作多个元件

| 功能     | Windows | Mac OS X |
|:---------|:--------|:---------|
| 左对齐   | C-M-l   |          |
| 左右居中 | C-M-c   |          |
| 右对齐   | C-M-r   |          |
| 顶部对齐 | C-M-t   |          |
| 上下居中 | C-M-m   |          |
| 底部对齐 | C-M-b   |          |
| 组合     | C-g     |          |
